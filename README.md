# Overview

This pong game has 2 modes 1 ball for classic mode and 2 balls for hard mode. Then the left player can use w and s to move their
paddle and right player can use up and down arrow keys to move their paddle.

This pong game was my initial attempt to break into game development. I think this built a lot of confidence that I could build
a pixle art rpg game that could actually go to market.

# Development Environment

This game was written in VS code using the python arcade libraries.
